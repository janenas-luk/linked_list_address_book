 # C linked list address book example
 
 This example provides code, structure and functionality  on how the linked list address book task could be done.
 This code isn't perfect and can be fixed. 
 If you have any suggestions on how to make the code better, please let me know. 
 
 ## Building program
 To build the program use:
 ```bash
 make
 ```
 ## Cleanup
 To remove the builded files use:
 ```
 make clean
 ```
 
 ## Usage
 To start using the program you need to use the executable:
 ```
 ./address_book
 ```
 No additional parametres are available
